Listas, Etiquetas y Estilo

Aprendemos a crear listas en HTML y a darles estilo con CSS

  1. ol= Listas ordenadas
  2. reversed= empieza de atras hacia adelante
  3. type A= Lista con letras
  4. type I= Lista con numeros romanos
  5. ul= Listas desordenadas
Ejemplos:

  1. Argentina
  2. Brazil
  3. México
  4. Paraguay

  2. Brazil
  3. México
  4. Paraguay

Como crear una lista de descripción y sus componentes

  1. dl= Listas de descripción
  2. dt= Se introduce el título
  3. dd= Aca iría la descripción del título
Ejemplos:
Argentina
Es el Campeón del Mundo
Francia
Es el Subcampeón del Mundo
